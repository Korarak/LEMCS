ภาพรวม

LEMCS คืออะไร?

LEMCS (Loei Educational MindCare System) คือ แพลตฟอร์มดิจิทัล สำหรับคัดกรองและติดตามสุขภาพจิต นักเรียนระดับอนุบาล–มัธยมปลาย ของจังหวัดเลย ครอบคลุมทุกสังกัด

ออกแบบให้ นักเรียนทำแบบประเมินเองได้ บนมือถือ ขณะที่ครูแนะแนวและผู้บริหารเห็น ผลแบบ Real-time พร้อมระบบแจ้งเตือนอัตโนมัติภายในระบบทันทีที่พบความเสี่ยง

เครื่องมือประเมิน

แบบประเมินมาตรฐานสากล 3 ชุด

😰
ST-5

วัดระดับความเครียด

  • อายุเป้าหมาย: 15 ปีขึ้นไป
  • จำนวน: 5 ข้อคำถาม
  • คะแนน: 0–15
  • 4 ระดับ: ปกติ / เล็กน้อย / ปานกลาง / รุนแรง
  • ไม่มี Suicide Flag
  • มาตรฐาน: กรมสุขภาพจิต ไทย
😔
PHQ-A

ภาวะซึมเศร้าวัยรุ่น

  • อายุเป้าหมาย: 11–20 ปี
  • จำนวน: 9 ข้อหลัก + BQ1/BQ2
  • คะแนน: 0–27
  • 5 ระดับ: ไม่มี → รุนแรงมาก
  • ⚠️ Suicide Flag: Q9 ≥ 1 หรือ BQ = true
  • มาตรฐาน: Spitzer et al.
🧒
CDI

ภาวะซึมเศร้าเด็ก

  • อายุเป้าหมาย: 7–17 ปี
  • จำนวน: 27 ข้อ (ก/ข/ค)
  • คะแนน: 0–54
  • 2 ระดับ: ปกติ (<15) / ต้องดูแล (≥15)
  • ≥15 = แจ้งเตือนอัตโนมัติ
  • มาตรฐาน: Maria Kovacs 1985

💡 ระบบเลือกอัตโนมัติตามอายุ: อายุ 15–17 ปี → ครบ 3 ชุด  |  11–14 ปี → CDI + PHQ-A  |  7–10 ปี → CDI  |  ต่ำกว่า 7 ปี หรือไม่มีวันเกิด → ไม่แสดงแบบประเมิน

① StatsCards — 4 ตัวเลขหลักที่เห็นทันที:

  • นักเรียนทั้งหมด — จำนวนนักเรียนในระบบ ณ ปัจจุบัน
  • ทำแล้วในรอบนี้ — จำนวน + % coverage ของรอบสำรวจที่เปิดอยู่
  • ความเสี่ยง moderate+ — นักเรียนที่ต้องติดตามโดยด่วน
  • Alert รอดำเนินการ — เคสที่ยังไม่มีครู/ผู้บริหารรับไปดูแล

② RiskFunnelChart — Funnel 4 ขั้น (เส้นทางดูแลนักเรียน):

ลงทะเบียน ▶ ทำแล้ว ▶ พบความเสี่ยง ▶ ดูแลแล้ว
  • แสดงจำนวนนักเรียนที่ลดลงในแต่ละขั้น — เห็นทันทีว่ากระบวนการดูแลมีช่องว่างตรงไหน

③ InsightPanel — ข้อสรุปอัตโนมัติ 3 ข้อ:

  • โรงเรียนที่มีอัตราความเสี่ยงสูงสุด Top 3 อันดับ
  • อำเภอที่มีนักเรียนยังไม่ทำแบบประเมินมากที่สุด
  • แบบประเมินที่พบความเสี่ยงระดับ moderate+ มากที่สุด

④ AffiliationStudentStats — Bar Chart แยกสังกัด:

C

📈 วิเคราะห์เชิงลึก (Analytics) — 5 กราฟ

  • TrendChart — Line Chart แนวโน้ม 6 เดือนล่าสุด
    แกน X: เดือน (เช่น ต.ค.–มี.ค.)  |  แกน Y: จำนวนที่พบความเสี่ยง
    3 เส้นแยกสี: ST-5 · PHQ-A · CDI — เห็นว่าช่วงไหนมีความเสี่ยงพุ่งสูง (เช่น ช่วงสอบ)
  • MoMDeltaChart — Bar เปรียบเทียบเดือนนี้ vs เดือนก่อน
    Bar แดง = ความเสี่ยงเพิ่มขึ้น (น่าเป็นห่วง)  |  Bar เขียว = ลดลง (ดีขึ้น)
    หน่วย: % การเปลี่ยนแปลง เช่น PHQ-A +12% หรือ CDI −5%
  • SeverityChart — Doughnut สัดส่วน 4 ระดับความเสี่ยง
    ■ ปกติ (ไม่ต้องดำเนินการ)  ·  ■ เล็กน้อย (ให้คำแนะนำ)  ·  ■ ปานกลาง (ครูแนะแนวดูแล)  ·  ■ รุนแรง (ต้องดำเนินการทันที)
    ตรงกลาง Doughnut = total นักเรียนที่ประเมินแล้วทั้งหมด
  • AssessmentTypeChart — Stacked Bar แยกตามประเภทแบบประเมิน
    แกน X: ST-5 / PHQ-A / CDI  |  แต่ละ bar = สัดส่วน % ทุกระดับรวม 100%
    เห็นได้ทันทีว่าแบบประเมินไหนพบความเสี่ยงระดับรุนแรงมากที่สุด
  • RiskProgressBars — แถบ Progress 4 ระดับ พร้อมจำนวนและ %
    ปกติ: N คน (xx%)  ·  เล็กน้อย: N คน  ·  ปานกลาง: N คน  ·  รุนแรง: N คน

ความปลอดภัย

ความปลอดภัยและ PDPA Compliance

🔐

AES-256 Encryption

  • เลขบัตรประชาชนเข้ารหัสทุก record
  • ไม่ปรากฏใน URL, Log, หรือ plain text
  • Key ต้องยาว 32 ตัวอักษร (256-bit)
  • ระบบ crash ถ้า Key ไม่ครบ
✅

PDPA Consent

  • Modal บังคับก่อนทำแบบประเมินครั้งแรก
  • บันทึก + Timestamp ที่ยินยอม
  • ปฏิเสธ = ออกจากระบบทันที
  • ดูนโยบายฯ ได้จากหน้า Login
📋

Audit Log

  • บันทึกทุกการเข้าถึงข้อมูลนักเรียน
  • เก็บ user_id + timestamp + action
  • Admin ดู Audit Log ผ่านหน้า /admin/audit-logs
  • รองรับการตรวจสอบ PDPA
🛡️

Session Security

  • นักเรียน: JWT 60 นาที
  • Admin ทุกระดับ: JWT 8 ชั่วโมง
  • Rate limit 100 req/นาที ต่อ IP
🚫

No PII in URL

  • ชื่อ / เลขบัตร ไม่ปรากฏใน URL path
  • ทุก endpoint ใช้ UUID เท่านั้น
  • ป้องกัน expose ผ่าน Browser history
🔄

Backup อัตโนมัติ

  • pg_dump ทุกวัน เวลา 02:00
  • ส่งไป MinIO Object Storage
  • รองรับ Point-in-time Recovery

# LEMCS (Loei Educational MindCare System)

ระบบสำรวจและประเมินสุขภาพจิตนักเรียนระดับชั้นมัธยมศึกษาและอาชีวศึกษาในจังหวัดเลย (ST-5, PHQ-A, CDI) 
ตัวระบบถูกออกแบบสถาปัตยกรรมสำหรับรองรับผู้ใช้งานขนาดใหญ่และมีการดูแลความปลอดภัยด้านข้อมูล (PDPA) ขั้นสูง

---

## 🚀 วิธีการรันโปรเจกต์ (Quick Start)

เพื่อให้ระบบทำงานได้สมบูรณ์ แนะนำให้รันผ่าน **Docker** (ระบบประกอบด้วย Next.js, FastAPI, PostgreSQL, Redis, MinIO)

### 1. การตั้งค่าเริ่มต้น (ทำแค่ครั้งแรก)
ให้ทำการคัดลอกไฟล์ `.env.example` เป็น `.env` ด้วยคำสั่ง:
```bash
cp .env.example .env
```
> **⚠️ สำคัญมาก**: ต้องเปิดไฟล์ `.env` ขึ้นมาตั้งค่า `ENCRYPTION_KEY` ให้มีความยาว **32 ตัวอักษรพอดีเป๊ะ** (สำหรับ AES-256) มิฉะนั้น Backend จะดับทันทีตอนเริ่มรัน

### 2. รันระบบ (Development Mode)
โหมดนี้จะเปิดระบบทั้งหมดและรองรับ **Hot Reload** (แก้โค้ดแล้วอัปเดตทันที)
```bash
docker compose -f docker-compose.yml -f docker-compose.dev.yml up -d
```
> **ทิปส์:** หากคุณแก้โค้ดฝั่ง Backend แล้วระบบไม่อัปเดตเนื่องจากไม่ได้ทำ Volume ไว้ ให้ใช้คำสั่งบังคับ Rebuild ฝั่ง Backend:
> `docker compose up -d --build backend`

### 3. การเข้าใช้งาน
เมื่อระบบรันสมบูรณ์ สามารถเข้าใช้งานบริการต่างๆ ได้ตามลิงก์ด้านล่าง:
- **หน้าเว็บผู้ใช้งาน (Frontend):** http://localhost:3000
- **หน้า API Docs (Swagger):** http://localhost:8000/docs
- **ระบบฐานข้อมูลไฟล์ (MinIO):** http://localhost:9001
- **ระบบกราฟ (Grafana):** http://localhost:3001

### 🔑 บัญชีสำหรับการทดสอบ (Test Accounts)
ระบบจะทำการ Seed ข้อมูลทดสอบให้ตอนเริ่มรันครั้งแรก:
- **เข้าสู่ระบบผู้ดูแล (Admin):** เข้าผ่าย `/admin-login` 
  - *(ในโหมด Dev มีปุ่ม Quick-Fill ให้กดกรอกข้อมูลอัตโนมัติแล้วกดเข้าสู่ระบบได้เลย)*
  - Username: `admin` / Password: `password123`
- **เข้าสู่ระบบนักเรียน (Student):**
  - รหัสนักเรียน: `12345` / บัตรประชาชน: `1234567890123` / วันเกิด: `2000-01-01`

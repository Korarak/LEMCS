Scoring Reference

ST-5  ความเครียด (5 ข้อ, 0–15 คะแนน)

0 – 4ปกติ (normal)
5 – 7เครียดเล็กน้อย (mild)
8 – 9เครียดปานกลาง (moderate)
10 – 15เครียดมาก (severe)

PHQ-A  ซึมเศร้าวัยรุ่น (9 ข้อ, 0–27 คะแนน)

0 – 4ไม่มีอาการ (normal)
5 – 9เล็กน้อย (mild)
10 – 14ปานกลาง (moderate)
15 – 19รุนแรง (severe)
20 – 27รุนแรงมาก (very_severe)

CDI  ซึมเศร้าเด็ก (27 ข้อ, 0–54 คะแนน)

0 – 12ปกติ (normal)
13 – 19เล็กน้อย (mild)
20 – 29ปานกลาง (moderate)
30 – 54รุนแรง (severe)
* บางข้อใช้ inverted scoring

Relationships

จากไปยังCardinalityความหมาย

districts──→schools1 : Many1 เขต มีได้หลายโรงเรียน
schools──→students1 : Many1 โรงเรียน มีได้หลายนักเรียน
students──→assessments1 : Many1 นักเรียน ทำแบบประเมินได้หลายครั้ง
assessments──→alerts1 : 0–11 ผลประเมิน สร้างได้ 0 หรือ 1 alert
students──→alerts1 : Many1 นักเรียน มีได้หลาย alert (ตลอดเวลา)

users──→alerts.assigned_to1 : Many1 user รับผิดชอบได้หลาย alert
users──→notifications1 : Many1 user รับได้หลาย notification
users──→audit_logs1 : Many1 user สร้าง log ได้หลายรายการ

Data Hierarchy

affiliations  (สังกัด: สพฐ., อาชีวะ, เอกชน, กศน.)
  └── districts  (เขตพื้นที่: สพป.เลย เขต 1–3, สพม.เลย, ...)
        └── schools  (โรงเรียน / สถานศึกษา)
              └── students  (นักเรียน — AES-256 encrypted national_id)
                    └── assessments  (ผลประเมิน ST5 / PHQA / CDI)
                          └── alerts  (แจ้งเตือน → assigned_to user)

users  (ผู้ดูแลระบบ + student accounts)
  ├── → notifications  (แจ้งเตือนใน dashboard)
  └── → audit_logs     (PDPA — ทุกการเข้าถึงข้อมูล)

## ส่วนที่ 1 — ภาพรวมระบบ

LEMCS เป็นระบบประเมินสุขภาพจิตสำหรับนักเรียน K–12 จังหวัดเลย ครอบคลุมนักเรียนกว่า 100,000 คน รองรับแบบประเมิน 3 ชุด:

| แบบประเมิน | กลุ่มเป้าหมาย (อายุ) | วัตถุประสงค์ |
|---|---|---|
| **ST-5** | อายุ **15 ปีขึ้นไป** | ประเมินความเครียด |
| **PHQ-A** | อายุ **11–20 ปี** | ประเมินภาวะซึมเศร้าวัยรุ่น |
| **CDI** | อายุ **7–17 ปี** | ประเมินภาวะซึมเศร้าในเด็กและวัยรุ่น |

> **กฎการเลือกแบบประเมิน (อิงอายุคำนวณจากวันเกิด):**
> - อายุ **7–10 ปี** → CDI เท่านั้น
> - อายุ **11–14 ปี** → CDI + PHQ-A
> - อายุ **15–17 ปี** → ST-5 + CDI + PHQ-A (ทั้ง 3 แบบ)
> - อายุ **18–20 ปี** → ST-5 + PHQ-A
> - อายุ **21 ปีขึ้นไป** → ST-5 เท่านั้น
> - อายุ **< 7 ปี** หรือไม่มีข้อมูลวันเกิด → ไม่แสดงแบบประเมิน

## ส่วนที่ 2 — ขั้นตอนการใช้งานของนักเรียน (Student Workflow)

### 2.1 การเข้าสู่ระบบ

นักเรียนเข้าสู่ระบบด้วย 3 ข้อมูลพร้อมกัน:
1. **รหัสนักเรียน** (Student Code)
2. **วันเกิด** (ใช้คำนวณอายุเพื่อเลือกแบบประเมินที่เหมาะสม)
3. **เลขบัตรประจำตัวประชาชน** (เข้ารหัส AES-256 ตาม พ.ร.บ. PDPA)

> **ข้อมูลทดสอบ (Seeded):** รหัส `12345` / วันเกิด `2000-01-01` / เลขบัตร `1234567890123`

### 2.2 หน้าจอยินยอม PDPA (ครั้งแรกเท่านั้น)

ก่อนทำแบบประเมินครั้งแรก ระบบแสดง **ConsentModal** บังคับ — นักเรียนต้องอ่านและติ๊กยืนยัน จึงจะดำเนินการต่อได้ ข้อมูลที่แจ้ง:
- ผลการประเมินสุขภาพจิต (ST-5, PHQ-A, CDI)
- ข้อมูลส่วนตัวพื้นฐานตามที่โรงเรียนให้ไว้
- ใช้เพื่อติดตามดูแลสุขภาพจิต และรายงานต่อครูแนะแนว/ผู้บริหาร
- เก็บรักษาภายใต้ พ.ร.บ. คุ้มครองข้อมูลส่วนบุคคล พ.ศ. 2562

### 2.3 หน้า Dashboard

หลังเข้าสู่ระบบ นักเรียนเห็น:
- **แบบประเมินที่เปิดให้ทำ** — รายการที่ระบบ/ครูเปิดให้ทำได้ในขณะนั้น
- **สุขภาพจิตของคุณ** — กราฟ Wellness Score จากประวัติการทำ
- **ประวัติการเข้าร่วม** — รายการผลประเมินทั้งหมดในอดีต

### 2.4 หน้าทำแบบประเมิน

- แสดงทีละ 1 ข้อ พร้อม Progress Bar
- นักเรียนเลือกคำตอบ → ระบบเลื่อนไปข้อถัดไปอัตโนมัติ (animation 350ms)
- มีปุ่ม **ย้อนกลับ** (แก้คำตอบข้อก่อนหน้าได้)
- **Auto-save** ทุก 30 วินาที (กรณีเน็ตหลุดหรือปิดแอปโดยไม่ตั้งใจ)
- เมื่อตอบครบทุกข้อ ปุ่ม **"ส่งคำตอบและประเมินผล"** จะเปิดใช้งาน
- มีหน้าต่างยืนยันก่อนส่ง — "หลังจากส่งแล้วจะไม่สามารถแก้ไขคำตอบได้"

### 2.5 หน้าผลการประเมิน

แสดงผลลัพธ์ทันทีหลังส่ง:
1. **คะแนนตัวเลข** (แสดงใหญ่)
2. **ป้ายระดับ** (Badge ระบายสี — ดูตาราง Severity ด้านล่าง)
3. **คำแนะนำเบื้องต้น** (ขึ้นกับระดับความรุนแรง)
4. **ทรัพยากรช่วยเหลือวิกฤต** (แสดงเมื่อ suicide_risk = true หรือระดับ moderate/severe/very_severe/clinical)

## ส่วนที่ 3 — แบบประเมินความเครียด ST-5

### 3.1 ที่มาและมาตรฐานอ้างอิง

ST-5 (Stress Test 5) พัฒนาโดยกรมสุขภาพจิต กระทรวงสาธารณสุข ประเทศไทย ใช้คัดกรองความเครียดในประชากรทั่วไปรวมถึงนักเรียน

### 3.2 ข้อคำถาม (5 ข้อ)

| ข้อ | คำถาม | ตัวเลือก (คะแนน) |
|---|---|---|
| Q1 | คุณรู้สึกไม่สบายใจ กังวล หรือเครียดบ้างไหม? | ไม่เลย(0) / บางครั้ง(1) / บ่อยครั้ง(2) / เกือบทุกวัน(3) |
| Q2 | คุณรู้สึกหงุดหงิด รำคาญใจ หรือโกรธง่ายบ้างไหม? | ไม่เลย(0) / บางครั้ง(1) / บ่อยครั้ง(2) / เกือบทุกวัน(3) |
| Q3 | คุณรู้สึกเหนื่อยในการใช้ชีวิตหรือรู้สึกว่าตัวเองมีภาระมากเกินไปบ้างไหม? | ไม่เลย(0) / บางครั้ง(1) / บ่อยครั้ง(2) / เกือบทุกวัน(3) |
| Q4 | คุณมีปัญหาการนอนหลับ เช่น นอนไม่หลับ ตื่นกลางดึก หรือหลับมากผิดปกติบ้างไหม? | ไม่เลย(0) / บางครั้ง(1) / บ่อยครั้ง(2) / เกือบทุกวัน(3) |
| Q5 | คุณรู้สึกว่าความเครียดส่งผลต่อการเรียน การใช้ชีวิต หรือความสัมพันธ์กับคนรอบข้างบ้างไหม? | ไม่เลย(0) / บางครั้ง(1) / บ่อยครั้ง(2) / เกือบทุกวัน(3) |

### 3.3 การคำนวณคะแนน

```
คะแนนรวม = Q1 + Q2 + Q3 + Q4 + Q5
ช่วงคะแนน: 0–15
```

**Source code (backend/app/services/scoring/st5.py):**
```python
score = sum(int(responses.get(f"q{i}", 0)) for i in range(1, 6))
score = max(0, min(15, score))  # clamp ไม่ให้เกิน 0–15
```

### 3.4 เกณฑ์แปลผล

| คะแนน | ระดับ (code) | ป้ายแสดงผล (ไทย) | สีแสดงผล |
|---|---|---|---|
| 0–4 | `normal` | ปกติ | เขียว (success) |
| 5–7 | `mild` | เครียดเล็กน้อย | น้ำเงิน (info) |
| 8–11 | `moderate` | เครียดปานกลาง | เหลือง (warning) |
| 12–15 | `severe` | เครียดสูง | แดง (error) |

**Source code:**
```python
if score <= 4:   level = "normal"
elif score <= 7:  level = "mild"
elif score <= 11: level = "moderate"
else:             level = "severe"
```

### 3.5 การแจ้งเตือนความเสี่ยง

- **ST-5 ไม่มี Suicide Risk flag** (suicide_risk = False เสมอ)
- ระดับ `severe` → ระบบแสดงคำแนะนำให้พบครูแนะแนวโดยเร็ว และแสดง CrisisResources ที่หน้าผล

### 3.6 คำแนะนำที่ระบบแสดงต่อนักเรียน

| ระดับ | คำแนะนำ |
|---|---|
| normal | "คุณมีระดับความเครียดอยู่ในเกณฑ์ปกติ" / "ดูแลสุขภาพจิตต่อไป ออกกำลังกาย พักผ่อนให้เพียงพอ" |
| mild | "คุณมีความเครียดเล็กน้อย ไม่ต้องกังวล" / "ลองพักผ่อน ทำกิจกรรมที่ชอบ หรือคุยกับเพื่อนสนิท" |
| moderate | "คุณมีระดับความเครียดปานกลาง" / "แนะนำให้พูดคุยกับครูแนะแนวหรือผู้ปกครอง" / "ฝึกหายใจลึกๆ และจัดการเวลาให้ดีขึ้น" |
| severe | "คุณมีระดับความเครียดสูง" / "⚠️ ควรพบครูแนะแนวโดยเร็วที่สุด" / "หากรู้สึกไม่ไหว โทรสายด่วนสุขภาพจิต 1323 (24 ชั่วโมง)" |

## ส่วนที่ 4 — แบบประเมินภาวะซึมเศร้าวัยรุ่น PHQ-A

### 4.1 ที่มาและมาตรฐานอ้างอิง

PHQ-A (Patient Health Questionnaire – Adolescent) เป็นฉบับมาตรฐานของสถาบันสุขภาพจิตเด็กและวัยรุ่นราชนครินทร์ พ.ศ. 2561 ดัดแปลงจาก PHQ-9 สำหรับวัยรุ่น กลุ่มเป้าหมาย: **อายุ 11–20 ปี**

### 4.2 ข้อคำถาม

**ส่วนหลัก Q1–Q9** (ตัวเลือก: ไม่มีเลย=0 / มีบางวัน=1 / มีมากกว่า 7 วัน=2 / มีแทบทุกวัน=3)

| ข้อ | คำถาม | หมายเหตุทางคลินิก |
|---|---|---|
| Q1 | รู้สึกซึมเศร้า หงุดหงิด หรือสิ้นหวัง | Depressed mood / Irritability |
| Q2 | เบื่อ ไม่ค่อยสนใจหรือไม่เพลิดเพลินเวลาทำสิ่งต่างๆ | Anhedonia |
| Q3 | นอนหลับยาก หรือหลับๆ ตื่นๆ หรือหลับมากเกินไป | Sleep disturbance |
| Q4 | รู้สึกเหนื่อยล้า หรือไม่ค่อยมีพลัง | Fatigue / Energy loss |
| Q5 | ไม่อยากอาหาร น้ำหนักลด หรือกินมากกว่าปกติ | Appetite change |
| Q6 | รู้สึกแย่กับตัวเอง หรือรู้สึกว่าตัวเองล้มเหลว หรือทำให้ตัวเองหรือครอบครัวผิดหวัง | Worthlessness / Guilt |
| Q7 | จดจ่อกับสิ่งต่างๆ ได้ยาก เช่น ทำการบ้าน อ่านหนังสือ หรือดูโทรทัศน์ | Concentration difficulty |
| Q8 | พูดหรือทำอะไรช้าลงจนคนอื่นสังเกตเห็นได้ หรือกระสับกระส่าย/กระวนกระวายจนต้องเคลื่อนไหวมากกว่าปกติ | Psychomotor changes |
| **Q9** | **คิดว่าถ้าตายไปเสียจะดีกว่า หรือคิดจะทำร้ายตัวเองด้วยวิธีใดวิธีหนึ่ง** | **Suicidal ideation — ตัวกระตุ้น Suicide Flag** |

**ส่วน Bonus Questions (ไม่นับคะแนน) — ประเมินความเสี่ยงฆ่าตัวตาย**

| ข้อ | คำถาม | ตัวเลือก |
|---|---|---|
| BQ1 | ใน 1 เดือนที่ผ่านมา มีช่วงไหนที่คุณมีความคิดอยากตาย หรือไม่อยากมีชีวิตอยู่อย่างจริงจังหรือไม่? | ไม่มี / มี |
| BQ2 | ตลอดชีวิตที่ผ่านมา คุณเคยพยายามที่จะทำให้ตัวเองตาย หรือลงมือฆ่าตัวตายหรือไม่? | ไม่เคย / เคย |

> BQ1 และ BQ2 **ไม่ถูกนำมาคิดคะแนนรวม** แต่ใช้ตรวจ Suicide Risk เท่านั้น

### 4.3 การคำนวณคะแนน

```
คะแนนรวม = Q1 + Q2 + Q3 + Q4 + Q5 + Q6 + Q7 + Q8 + Q9
ช่วงคะแนน: 0–27
```

**Source code (backend/app/services/scoring/phqa.py):**
```python
score = sum(int(responses.get(f"q{i}", 0)) for i in range(1, 10))
score = max(0, min(27, score))
```

### 4.4 เกณฑ์แปลผล

| คะแนน | ระดับ (code) | ป้ายแสดงผล (ไทย) | สีแสดงผล |
|---|---|---|---|
| 0–4 | `none` | ไม่มีภาวะซึมเศร้า | เขียว (success) |
| 5–9 | `mild` | เล็กน้อย | น้ำเงิน (info) |
| 10–14 | `moderate` | ปานกลาง | เหลือง (warning) |
| 15–19 | `severe` | มาก | แดง (error) |
| 20–27 | `very_severe` | รุนแรง | แดง (error) |

**Source code:**
```python
if score <= 4:    level = "none"
elif score <= 9:  level = "mild"
elif score <= 14: level = "moderate"
elif score <= 19: level = "severe"
else:             level = "very_severe"
```

### 4.5 การตรวจ Suicide Risk

**Suicide flag ถูกกำหนดเป็น True เมื่อเข้าเงื่อนไขใดเงื่อนไขหนึ่ง:**

```python
suicide_risk = (
    int(responses.get("q9", 0)) >= 1    # Q9 ตอบ ≥ 1 (มีบางวันขึ้นไป)
    or bool(responses.get("bq1", False)) is True   # BQ1 ตอบ "มี"
    or bool(responses.get("bq2", False)) is True   # BQ2 ตอบ "เคย"
)
```

**สรุปเงื่อนไข Suicide Risk (PHQ-A):**

| เงื่อนไข | ค่าที่ trigger |
|---|---|
| Q9 "คิดจะทำร้ายตัวเอง" | ตอบ ≥ 1 (มีบางวัน, มีมากกว่า 7 วัน, หรือมีแทบทุกวัน) |
| BQ1 ความคิดอยากตายใน 1 เดือนที่ผ่านมา | ตอบ "มี" |
| BQ2 เคยพยายามฆ่าตัวตาย | ตอบ "เคย" |

### 4.6 คำแนะนำที่ระบบแสดงต่อนักเรียน

| ระดับ | คำแนะนำ |
|---|---|
| none | "ไม่พบสัญญาณของภาวะซึมเศร้า 😊" / "ดูแลสุขภาพจิตต่อไป นอนหลับพักผ่อนให้เพียงพอ" |
| mild | "มีอาการเล็กน้อย ลองออกกำลังกายและพูดคุยกับคนที่ไว้ใจ" |
| moderate | "แนะนำให้พบครูแนะแนวหรือนักจิตวิทยา" / "อย่าอยู่คนเดียวนาน ทำกิจกรรมร่วมกับเพื่อน" |
| severe | "⚠️ ควรพบผู้เชี่ยวชาญด้านสุขภาพจิตโดยเร็ว" / "โทรสายด่วน 1323 ได้ตลอด 24 ชั่วโมง" |
| very_severe | "🚨 กรุณาพบแพทย์หรือนักจิตวิทยาทันที" / "โทรสายด่วนสุขภาพจิต 1323 หรือแจ้งผู้ปกครองทันที" |

## ส่วนที่ 5 — แบบประเมินภาวะซึมเศร้าในเด็ก CDI

### 5.1 ที่มาและมาตรฐานอ้างอิง

CDI (Children's Depression Inventory) พัฒนาโดย Maria Kovacs (1985) ฉบับแปลไทย ใช้สำหรับเด็กและวัยรุ่นอายุ **7–17 ปี** ตามมาตรฐานสากล (MHS, MHS CDI 2; ScienceDirect; PMC) ในระบบนี้ใช้ CDI สำหรับนักเรียนอายุ **7–17 ปี**

### 5.2 ข้อคำถามทั้ง 27 ข้อ

แต่ละข้อมี 3 ตัวเลือก (ก / ข / ค) — แต่**คะแนนไม่เท่ากันในทุกข้อ** ขึ้นกับกลุ่มของข้อนั้น:

**กลุ่ม A (Group A) — คะแนน: ก=0, ข=1, ค=2** (ตรงทิศทาง — ค รุนแรงที่สุด)

ข้อที่อยู่ในกลุ่ม A: **1, 3, 4, 6, 9, 12, 14, 17, 19, 20, 22, 23, 26, 27** (14 ข้อ)

**กลุ่ม B (Group B) — คะแนน: ก=2, ข=1, ค=0** (กลับทิศทาง — ก รุนแรงที่สุด)

ข้อที่อยู่ในกลุ่ม B: **2, 5, 7, 8, 10, 11, 13, 15, 16, 18, 21, 24, 25** (13 ข้อ)

**Source code (backend/app/services/scoring/cdi.py):**
```python
CDI_GROUP_A = {1, 3, 4, 6, 9, 12, 14, 17, 19, 20, 22, 23, 26, 27}
CDI_GROUP_B = {2, 5, 7, 8, 10, 11, 13, 15, 16, 18, 21, 24, 25}
```

### 5.6 การตรวจ Suicide Risk (CDI)

**Suicide flag ถูกกำหนดเป็น True เมื่อ:**

```python
suicide_risk = responses.get("q9", "ก") == "ค"
# ข้อ 9 ตอบ "ค. ฉันอยากตาย"
```

| ตัวเลือก Q9 | ความหมาย | Suicide Risk |
|---|---|---|
| ก | ฉันไม่ได้คิดเรื่องตาย | False |
| ข | ฉันคิดเรื่องตายแต่ไม่อยากทำ | **False** |
| ค | ฉันอยากตาย | **True** |

> **ข้อสังเกต:** CDI Q9 ตอบ "ข" (คิดเรื่องตาย แต่ไม่อยากทำ) ยังไม่ถือว่าเป็น Suicide Risk ตามการตั้งค่าของระบบ ต่างจาก PHQ-A ที่ Q9 ≥ 1 trigger แล้ว

### 5.7 คำแนะนำที่ระบบแสดงต่อนักเรียน

| ระดับ | คำแนะนำ |
|---|---|
| normal | "ไม่พบสัญญาณของภาวะซึมเศร้า ดูแลสุขภาพต่อไป 😊" |
| clinical | "⚠️ พบสัญญาณที่ควรได้รับการดูแลเพิ่มเติม" / "ครูแนะแนวจะติดต่อมาเพื่อพูดคุยกับคุณเร็วๆ นี้" / "หากรู้สึกไม่สบายใจ คุยกับครูหรือผู้ปกครองได้เลย" |

## ส่วนที่ 6 — ระบบเตือนภัยวิกฤต (Suicide Risk Protocol)

### 6.1 เงื่อนไขกระตุ้น

| แบบประเมิน | เงื่อนไข suicide_risk = True |
|---|---|
| ST-5 | **ไม่มี** (False เสมอ) |
| PHQ-A | Q9 ≥ 1 **หรือ** BQ1 = "มี" **หรือ** BQ2 = "เคย" |
| CDI | Q9 = "ค" (ฉันอยากตาย) |

### 6.2 สิ่งที่ระบบดำเนินการเมื่อ suicide_risk = True

1. **ทันทีบนหน้าจอนักเรียน** — แสดงแบนเนอร์เร่งด่วน (สีแดง + animation กระดิก) พร้อมข้อความ "🚨 เรื่องด่วน ขอความช่วยเหลือทันที" และปุ่มโทรหาสายด่วน **1323** และ **1387** แบบกดโทรได้เลย
2. **สร้าง Critical Alert** ในระบบ → แจ้งเตือนครูแนะแนว/ผู้บริหาร ผ่าน LINE Notify → Email SMTP → การแจ้งเตือนในแอป (ตามลำดับความสำคัญ)
3. **การแจ้งเตือนส่งแบบ Synchronous** (ไม่ผ่าน queue) เพื่อความรวดเร็ว

### 6.3 ทรัพยากรสายด่วนที่แสดงต่อนักเรียน

| สายด่วน | หมายเลข | เวลา |
|---|---|---|
| สายด่วนสุขภาพจิต | **1323** | 24 ชั่วโมง (โทรฟรี) |
| สายด่วนช่วยเหลือเด็ก | **1387** | 24 ชั่วโมง |
| กรมสุขภาพจิต | **02-149-5555** | (ข้อมูลเพิ่มเติมในระบบ) |

### 6.4 เงื่อนไขแสดง CrisisResources (ไม่ใช่ Suicide แต่ระดับสูง)

ระบบแสดงแบนเนอร์ช่วยเหลือ (สีเหลือง ไม่ urgent) ในกรณีดังนี้ (แม้ suicide_risk = False):
- PHQ-A ระดับ `moderate`, `severe`, `very_severe`
- CDI ระดับ `clinical`
- ST-5 ระดับ `severe`

### 8.2 ข้อสังเกตเพิ่มเติมจากการวิเคราะห์โค้ด

1. **CDI Q9 ข้อ "ข" ไม่ trigger Suicide Risk** — ระบบกำหนดว่า "คิดเรื่องตายแต่ไม่อยากทำ" ไม่ใช่ความเสี่ยง ต่างจาก PHQ-A ที่ Q9 ≥ 1 ก็ trigger แล้ว ขอให้จิตแพทย์พิจารณาว่าเหมาะสมสำหรับเด็กและวัยรุ่นอายุ 7–17 ปี หรือไม่

2. **PHQ-A BQ1/BQ2 trigger suicide_risk ได้โดยอิสระจาก Q9** — หมายความว่าแม้คะแนนรวมอยู่ใน "none" (0–4) แต่ถ้าตอบ BQ1/BQ2 ว่ามีความคิดอยากตาย ก็จะได้รับการแจ้งเตือนวิกฤตทันที

3. **ST-5 ไม่มี Suicide Risk flag เลย** — เป็นไปตามธรรมชาติของแบบประเมินความเครียด แต่ควรพิจารณาว่ามีกรณีที่ ST-5 ควรตรวจ suicide risk บ้างหรือไม่
